
# Install
## Raspberry
1. Install raspbian
2. Tinkerforge daemon
<code>
  wget http://download.tinkerforge.com/tools/brickd/linux/brickd_linux_latest_armhf.deb
  sudo dpkg -i brickd_linux_latest_armhf.deb
  </code>
3. Node red
4. Inluxdb + chronograf
